# Especificaciones del Proyecto

Este directorio contiene las especificaciones, tareas y documentación del proyecto "El Parcial Cujae Edition".

## Estructura

- **UI/** - Especificaciones y mockups de interfaz de usuario
- **Gameplay/** - Documentación de mecánicas y reglas de juego
- **Assets/** - Especificaciones de recursos
  - **Sprites/** - Especificaciones de sprites
  - **Audio/** - Especificaciones de audio y música
  - **Background/** - Especificaciones de fondos

## Uso

Agregar archivos Markdown, imágenes de mockups, y documentación en cada subcarpeta según corresponda.
