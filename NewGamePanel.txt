● Paso completo: NewGamePanel

  1. Crear el contenedor

  1. Clic derecho sobre Canvas → Create Empty → renómbralo a NewGamePanel
  2. En Rect Transform → clic en el cuadrito de anclaje → mantén Alt → selecciona stretch-stretch
  3. Add Component → Image
    - Color: #000000, Alpha: 200

  ---
  2. Crear el título

  1. Clic derecho sobre NewGamePanel → UI → Text - TextMeshPro → renómbralo a TxtTituloSlots
  2. Texto: Seleccionar Partida
  3. Font Size: 48
  4. Alignment: centro
  5. En Rect Transform:
    - Anchor: top-center
    - Pos Y: -60
    - Width: 600, Height: 70

  ---
  3. Crear el Slot 0

  Botón del slot:
  1. Clic derecho sobre NewGamePanel → UI → Button - TextMeshPro → renómbralo a BtnSlot0
  2. En Rect Transform:
    - Anchor: middle-center
    - Pos X: -50, Pos Y: 100
    - Width: 400, Height: 80
  3. Add Component → MenuButton
  4. Expande BtnSlot0 → selecciona el hijo Text (TMP) → renómbralo a TxtSlot0
    - Texto: Slot 1 - Vacío
    - Font Size: 24

  Botón eliminar:
  1. Clic derecho sobre NewGamePanel → UI → Button - TextMeshPro → renómbralo a BtnDeleteSlot0
  2. En Rect Transform:
    - Anchor: middle-center
    - Pos X: 270, Pos Y: 100 (a la derecha del slot)
    - Width: 150, Height: 80
  3. Expande → hijo Text (TMP) → texto: Eliminar
    - Font Size: 20

  ---
  4. Crear el Slot 1

  Botón del slot:
  1. Clic derecho sobre NewGamePanel → UI → Button - TextMeshPro → renómbralo a BtnSlot1
  2. En Rect Transform:
    - Pos X: -50, Pos Y: 0
    - Width: 400, Height: 80
  3. Add Component → MenuButton
  4. Expande → hijo Text (TMP) → renómbralo a TxtSlot1
    - Texto: Slot 2 - Vacío

  Botón eliminar:
  1. UI → Button - TextMeshPro → renómbralo a BtnDeleteSlot1
  2. Pos X: 270, Pos Y: 0, Width: 150, Height: 80
  3. Texto: Eliminar

  ---
  5. Crear el Slot 2

  Botón del slot:
  1. UI → Button - TextMeshPro → renómbralo a BtnSlot2
  2. Pos X: -50, Pos Y: -100, Width: 400, Height: 80
  3. Add Component → MenuButton
  4. Hijo Text (TMP) → renómbralo a TxtSlot2
    - Texto: Slot 3 - Vacío

  Botón eliminar:
  1. UI → Button - TextMeshPro → renómbralo a BtnDeleteSlot2
  2. Pos X: 270, Pos Y: -100, Width: 150, Height: 80
  3. Texto: Eliminar

  ---
  6. Crear el botón Volver

  1. Clic derecho sobre NewGamePanel → UI → Button - TextMeshPro → renómbralo a BtnBackNewGame
  2. Texto: Volver
  3. En Rect Transform:
    - Anchor: bottom-center
    - Pos Y: 50
    - Width: 200, Height: 50
  4. Add Component → MenuButton
  5. En Button → On Click():
    - Clic en +
    - Arrastra TitleScreenController
    - Función → TitleScreenManager → GoBackToMain

  ---
  7. Agregar el script NewGamePanel

  1. Selecciona el GameObject NewGamePanel
  2. Add Component → busca NewGamePanel → agrégalo
  3. En el Inspector:

  Slot Labels → clic en la flechita, pon Size: 3

  ┌───────────┬───────────────┐
  │ Elemento  │ Qué arrastrar │
  ├───────────┼───────────────┤
  │ Element 0 │ TxtSlot0      │
  ├───────────┼───────────────┤
  │ Element 1 │ TxtSlot1      │
  ├───────────┼───────────────┤
  │ Element 2 │ TxtSlot2      │
  └───────────┴───────────────┘

  Delete Buttons → Size: 3

  ┌───────────┬────────────────┐
  │ Elemento  │ Qué arrastrar  │
  ├───────────┼────────────────┤
  │ Element 0 │ BtnDeleteSlot0 │
  ├───────────┼────────────────┤
  │ Element 1 │ BtnDeleteSlot1 │
  ├───────────┼────────────────┤
  │ Element 2 │ BtnDeleteSlot2 │
  └───────────┴────────────────┘

  ---
  8. Conectar los onClick de cada botón

  Botones de slot → arrastra NewGamePanel al campo de objeto:

  ┌──────────┬─────────────────────────────┬─────────────────┐
  │  Botón   │           Función           │ Parámetro (int) │
  ├──────────┼─────────────────────────────┼─────────────────┤
  │ BtnSlot0 │ NewGamePanel.OnSlotSelected │ 0               │
  ├──────────┼─────────────────────────────┼─────────────────┤
  │ BtnSlot1 │ NewGamePanel.OnSlotSelected │ 1               │
  ├──────────┼─────────────────────────────┼─────────────────┤
  │ BtnSlot2 │ NewGamePanel.OnSlotSelected │ 2               │
  └──────────┴─────────────────────────────┴─────────────────┘

  Botones eliminar → arrastra NewGamePanel:

  ┌────────────────┬───────────────────────────┬─────────────────┐
  │     Botón      │          Función          │ Parámetro (int) │
  ├────────────────┼───────────────────────────┼─────────────────┤
  │ BtnDeleteSlot0 │ NewGamePanel.OnDeleteSlot │ 0               │
  ├────────────────┼───────────────────────────┼─────────────────┤
  │ BtnDeleteSlot1 │ NewGamePanel.OnDeleteSlot │ 1               │
  ├────────────────┼───────────────────────────┼─────────────────┤
  │ BtnDeleteSlot2 │ NewGamePanel.OnDeleteSlot │ 2               │
  └────────────────┴───────────────────────────┴─────────────────┘

  Importante: Cuando selecciones la función, Unity muestra un campo numérico abajo. Escribe el número 0, 1 o 2 ahí.

  ---
  9. Conectar al TitleScreenController

  1. Selecciona TitleScreenController
  2. En el script TitleScreenManager:

  ┌─────────────────────────┬───────────────┐
  │          Campo          │ Qué arrastrar │
  ├─────────────────────────┼───────────────┤
  │ New Game Panel          │ NewGamePanel  │
  ├─────────────────────────┼───────────────┤
  │ New Game First Selected │ BtnSlot0      │
  └─────────────────────────┴───────────────┘

  ---
  10. Desactivar el panel

  1. Selecciona NewGamePanel → desmarca el checkbox: ☐ NewGamePanel

  ---
  11. Probar

  1. Ctrl+S para guardar
  2. Dale Play ▶
  3. Toca Nuevo Juego o pulsa N
  4. Verifica:
    - Los 3 slots aparecen mostrando "Vacío" ✓
    - Al tocar un slot vacío, se guarda la fecha y carga la escena Gameplay ✓
    - Los botones Eliminar de slots vacíos están ocultos ✓
    - Esc o Volver regresa al MainPanel ✓

  Nota: Al tocar un slot te dará error porque la escena "Gameplay" no existe aún. Eso es normal. Cuando la creen, funcionará automáticamente.
